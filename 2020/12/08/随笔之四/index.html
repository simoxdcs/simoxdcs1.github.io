# 2020-12-8 星期二 -3。C~0。C 多云

又过去了很久啊。。总是懒得去写，这几天呢，学完了 MarkDown 的基本语法，还有 Typora 编辑器的基本使用，前一段时间还学了 VScode 的简单使用，还有一些插件的安装。但是不知打怎么回事，这几天一直有点头疼… 而且早上总是昏沉沉的起不来。。很奇怪。。快要期末考试了，应该就快可以回家了，不过这次我妈让我先去外公那看一看，毕竟很久没有去了。所以可能要放了假再过个 5、6 天才会回去。这段时间要背一背单词了。还要把高数好好复习一下。。。回去还要给 101 的下一届小崽子们作宣讲。。尽量多学点东西吧。。前天晚上还搭建好了一个自己的在 github 上面的图床。。以后就可以加入自己想要的图片了。。嘿嘿。但是就是 bgm 的那块始终有问题。。一直改不好。。啥时候想起来了再看看吧。。前两天做梦竟然梦到我有女朋友了。。之前还参加了个什么三行情书的比赛。。

月，洒在了华山畿。

风，停在了爱尔兰。

你，驻在了我心头。

好累。。不知道怎么回事。。遇事不决，可问春风。
